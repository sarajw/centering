How to center anything with CSS

Here we go, hopefully this will be
a clear cut guide to centering stuff.

Already I'm gritting my teeth because of my British English wanting to 'centre' everything instead, but there you go.

This is how it works:

  • Everything that's centered on this page is centred with inline styles.
  • When hovering over a centered element, the styles in use will show.
  • If you know how to use the dev tools in your browser, use that too!

This block of text is within the <main> element, which is itself within a CSS grid. Its parent (<body>) is a CSS grid.
